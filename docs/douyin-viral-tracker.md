# 抖音爆款视频追踪系统 - 详细指南

## 📊 追踪指标说明

### 1. 视频基本信息
- **视频标题**: 原始视频标题
- **发布日期**: YYYY-MM-DD格式
- **博主名称**: 内容创作者ID/昵称
- **视频类型**: 翻盘故事、逆袭、反转等分类

### 2. 数据指标 (统计时间点)
- **播放量**: 视频总播放数
- **点赞数**: 获赞数量
- **评论数**: 评论数量
- **分享数**: 分享次数
- **收藏数**: 收藏次数
- **互动率**: (点赞+评论+分享+收藏)/播放量
- **统计日期**: 数据统计的时间点

### 3. 内容分析
- **核心创意**: 视频的核心卖点是什么
- **翻盘元素**: 具体的翻盘/反转点
- **目标受众**: 主要受众群体描述
- **传播特点**: 为什么容易传播，传播优势

### 4. 成功要素拆解
- **热点利用**: 是否蹭热点，具体热点是什么
- **情感驱动**: 何种情感(感动、惊喜、愤怒、羡慕、幽默等)
- **视觉特色**: 特殊的画面表现、剪辑手法
- **文案强度**: 标题/正文的吸引力等级(1-5分)
- **音乐搭配**: 背景音乐风格

### 5. 参考链接
- **视频链接**: 完整的抖音视频URL
- **博主主页**: 完整的抖音账号链接

## 🎯 分析框架

### 维度1: 内容类型趋势
- 不同类型爆款视频的数量占比
- 各类型的平均播放量、互动率
- 类型流行趋势变化

### 维度2: 情感驱动对传播的影响
- 不同情感视频的互动率对比
- 情感组合的效果(单一vs混合)
- 目标受众与情感类型的匹配度

### 维度3: 博主特征分析
- 粉丝量与爆款率的关系
- 博主垂直度(专注领域)对传播的影响
- 更新频率与热度的关联

### 维度4: 时间规律
- 发布时间对传播速度的影响
- 季节性内容趋势
- 热点周期规律

### 维度5: 视觉&文案设计
- 视觉手法的有效性评分
- 标题类型分类(数字型、疑问型、对比型等)
- 文案长度与互动率的关系

### Python数据分析示例

```python
import pandas as pd
import json

# 读取CSV
df = pd.read_csv('data/viral-videos.csv')

# 基本统计
print(f"平均播放量: {df['播放量'].mean():,.0f}")
print(f"平均互动率: {(df['互动率'].mean()*100):.2f}%")

# 按类型分析
type_analysis = df.groupby('视频类型').agg({
    '播放量': 'mean',
    '点赞数': 'mean',
    '视频标题': 'count'
}).rename(columns={'视频标题': '视频数量'})
print(type_analysis)

# 情感驱动分析
emotion_performance = df.groupby('情感驱动')['播放量'].mean().sort_values(ascending=False)
print(emotion_performance)
```
