# 🎬 抖音爆款视频追踪系统

## 📌 项目简介

本项目用于系统地追踪、分析和研究**抖音平台"翻盘"相关爆款视频**，探索内容创意的成功规律和传播机制。

**主要目标：**
- 📊 建立爆款视频数据库
- 🔍 深度分析成功要素
- 💡 提炼可复制的创意模式
- 📈 发现内容趋势

---

## 🗂️ 项目结构

```
/
├── README.md (本文件)
├── docs/
│   └── douyin-viral-tracker.md          # 系统说明文档
├── data/
│   ├── viral-videos.csv                 # 爆款视频数据库 (主要数据)
│   ├── videos-analysis.json             # 详细分析数据 (JSON格式)
│   └── analysis-template.md             # 单个视频分析模板
└── reports/
    └── (月度分析报告存放处)
```

---

## 🚀 快速开始

### 1️⃣ 添加新视频
打开 `data/viral-videos.csv`，在最后一行添加新视频数据。

### 2️⃣ 深度分析
使用 `data/analysis-template.md` 作为模板进行详细分析。

### 3️⃣ 更新数据库
在 `data/videos-analysis.json` 中添加结构化数据。

### 4️⃣ 生成报告
汇总分析结果输出报告。

## 📈 分析维度

### 按内容类型分析
- 翻盘故事的特征
- 逆袭视频的模式
- 反转内容的传播力

### 按情感驱动分析
- 励志视频的转化率
- 惊喜反转的互动率
- 不同情感的传播速度

### 按博主维度分析
- 粉丝量与爆款率的关系
- 博主垂直度的影响
- 更新频率与传播效果

### 按时间维度分析
- 发布时间与传播速度
- 季节性内容趋势
- 热点周期规律
